10 Html Table A table in HTML consists of table cells inside rows and columns.
table It defines a table
tr It defines a row in a table
th If defines a header cell in a table
td It defines a cell in a table
Html table border It is used for border attribute of table
Colspan If we want a cell span more than one column,you can use the colspan attribute.It will divide one cell into multiple columns.
Rowspan If we want make a cell span more than one row,you can use the rowspan attribute .It will divide a cell into multiple columns.
Cell padding Cell padding in html is a property that specifies the space between the border of a table cell and its content in simplier terms.It defines the whitespace between the cell edge and the content inside the cell.
Cellspacing Cell spacing is the space between each cell,By default the space is set to 2 pixels.
Caption It defines a cell in a table
